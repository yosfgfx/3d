# 3D Rendering

موقع لإنشاء صور ثلاثية الأبعاد باستخدام الذكاء الاصطناعي

## المميزات

- واجهة مستخدم سهلة وبسيطة
- دعم اللغة العربية
- إمكانية تكبير النافذة وعرض ملء الشاشة
- تصميم متجاوب مع جميع الأجهزة

## التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript
- Hugging Face Spaces
